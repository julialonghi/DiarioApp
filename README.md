# DiarioApp

O aplicativo Diário possui o objetivo de servir como um refúgio aos seus usuários, de modo que possam escrever tudo que passa pelas suas mentes de maneira segura e acolhedora. O app possui uma interface simplificada, a fim de que todos possam utilizá-lo com facilidade.

Ao entrar no aplicativo, é exibida uma tela com "Meus Diários", onde todos os diários criados pelo usuário aparecem, assim, podendo existir diversos diários para diferentes contextos (Faculdade, Trabalho, etc). Para adicionar um novo, basta clicar no ícone de + que aparece no primeiro card na tela inicial e adicionar um nome a ele no PopUp que surge assim que o ícone é clicado. Para escrever em um dos diários, o usuário deve clicar no card desejado e acessar a TextField, digitar o que deseja e apertar o botão de Enviar, com o obejtivo de servir como um diário, as mensagens escritas ficam agrupadas por dia.
